Multilingual Portfolio

This project was built for a client as a personal portfolio.

The project was built using Express JS for the server and the Pug templating engine for the static content. Sass was used for the styling. i18n was used to provide multilingual support.

As the site only serves static content the goal was to include as few dependencies and use as little javascript as possible to ensure a fast load time and enjoyable experience to all visitors. Mobile usability and responsiveness for various device sizes was also a top priority.

Working with the client through the design process and then providing tweaks and changes during development to create a site they were excited about was an enjoyable and rewarding process.
